# Ejb
This project shows applying of EJB JPA Hibernate of Java EE Platform

# Prerequisites
- Java 1.8 or higher

# How to launch the project
Just navigate to http://localhost:8080/EjbHibernateRmi

# The project was launched on the WildFly-18.0.1
